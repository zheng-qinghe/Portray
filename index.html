Setup Your Local Engine

Connect to your local compute to power this demo.

The compute engine runs entirely on your machine.

  1. Install dependencies: pip install -r requirements.txt
  2. Start the engine: python -m uvicorn api:app --port 8080
  3. Wait for initialization (1000 profiles, ~2s)

API: http://127.0.0.1:8080
